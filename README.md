## üöÄ Getting Started

### Prerequisites
- Python 3.7 or higher
- Text editor or IDE (VS Code, PyCharm, etc.)

### Installation
1. Clone this repository:
   ```bash
   git clone https://github.com/khabzox/learning-py.git
   cd learning-py
   ```

2. Start with the first module:
   ```bash
   cd 01-Intro
   python main.py
   ```
